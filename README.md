## Introduction
Example code to demonstrate UT with gtest & gmock.

Details are available in https://zhuanlan.zhihu.com/p/712034767

## Dependency
* gtest
* cmake3.16+

## How to Use
### Download
```
$ git clone https://github.com/zenigame213/ex_math.git
```
### Build
```
$ cd ex_math
$ mkdir build
$ cd build
$ cmake .. -DWITH_UT=ON
$ make
```
### Test
```
$ ut/libmath_unit_test
```
